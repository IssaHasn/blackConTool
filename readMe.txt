اداة blackCon لفحص ال urls الخطيرة التي قد تسبب اتصال خطر وانا اسمي هذا "اتصال اسود"
تعمل عن طريق اخذ ال domain الاصلي من ال url وتبحث عن ال ip الخاص به
وفي نفس الوقت تتتبع مسار ال url لتعرف ip التي سوف يصل اليه الاتصال
ثم تقارن المصادر مع بعضها للتاكد من ان هذا ال url تابع لل domain الموجود ضمنه

حيث قد تجد url معين يحوي domain ل facebook مثلا ولكن الاتصال لا يتم مع ip تابع ل facebook وقد يكون ip تابع لمصدر خطر يهدف لسرق البيانات 

ملاحظة : اذا رأيت تشابه بال ip بشكل كبير ولكن يختلفان برقم واحد فقط فقط يكون هذا لان الشركة تمتلك عدة سيرفرات ولكن متصلة في شبكة واحدة
 او قد يكون بسبب البروتوكلول المدخل في ال link ان كان http او https



BlackCon is a tool for checking dangerous urls that may cause a dangerous connection, and I call this “black connection”.
It works by taking the original domain from the URL and searching for its IP
At the same time, you trace the URL path to know the IP that the connection will reach
Then compare the sources with each other to ensure that this URL belongs to the domain it contains

Where you may find a specific URL that contains a domain for Facebook, for example, but the connection does not occur with an IP belonging to Facebook, and the IP may be belonging to a dangerous source aiming to steal data.

Note: If you see that the IP address is very similar but differs by only one number, this is because the company owns several servers, but they are connected in one network.
  Or it may be due to the protocol entered in the link, whether it is http or https
